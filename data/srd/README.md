# SRD Data

This directory contains SRD (Systems Reference Document) content sourced from:
- https://api.open5e.com/
- https://www.dnd5eapi.co/api/

The initial files contain a representative sample of SRD content.
Run `npm run fetch-srd` to download the full SRD dataset.

## Files

- `spells.json` — SRD spells (sample; run fetch-srd for full list)
- `monsters.json` — SRD monsters (sample; run fetch-srd for full list)
- `magic-items.json` — SRD magic items (sample; run fetch-srd for full list)
